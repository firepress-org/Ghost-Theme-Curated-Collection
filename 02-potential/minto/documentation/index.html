Introduction

Introduction to Ghost

Firstly just few words about ghost and few facts that you should know before installation. We suggest you to read an introduction to how Ghost themes work. This documentation is written by ghost creators and it will help you to understand themes better, also it will help you to use themes more advance. Further more keep in mind that the ghost platform is still relatively new, this means that you should expect new features in future and a lot of updates.

Tools you'll need

To customize your theme you have to edit .hbs, .css and .js files. Usually you can edit them with your default text editor but we suggest you to use Sublime text or Brackets. These two applications are much more advanced versions of your default text editor. Installation and usage guides are provided with both of these pieces of software. Just pick one and enjoy.

Installation

Introduction to installation

There are few ways to install ghost. Installation depends on your hosting and setup. We will help you with installation on ghost hosting plan and separate hosting plan, please choose your setup and follow steps carefully.

Ghost hosting plan installation

When you have Ghost hosting plan then installation of Minto theme is very simple.

  1. Login to your Ghost account
  2. Now go to My Blogs section and click Edit next to the blog you wish to install the theme on.
  3. Under the Custom Theme section, select Choose File and browse your computer for the .zip file of Minto theme and press Save Settings.
  4. Finally Minto theme is installed on your ghost blog over default theme.

Separate hosting plan installation

If you have separate hosting plan then you have to follow this steps.

  1. Go to ghost theme directory ghost/content/themes
  2. Copy theme folder to ghost theme directory.
  3. Restart ghost and login to your dashboard.
  4. Go to settings and under themes select Minto theme, save settings.
  5. Now your theme is installed successfully.

Configuration

Configuration introduction

After installation you have to configure your theme, otherwise it will not work properly.

Blog details

First thing you have to do is to properly customize your blog title, description, logo image and everything else in your blog settings.

  1. Blog Description - Text inserted in this field will be shown at homepage on the top.
  2. Blog Logo - Browse your main blog logo here, we have provided you logo.psd file to generate logo such as default one. Logo.psd file can be found at design/logo.psd
  3. Posts Per Page - Set number of posts that will be shown at bottom of homepage.

Favicons

In /assets/images you can find 5 default minto theme icons and default favicon .ico icon. Follow these steps to change icons:

  1. Firstly go to design/logo.psd
  2. Edit that file if you want similiar logo/favicon as default logo/favicon. In the other hand if you have your own logo proceed to the next step.
  3. Now go to content/themes/minto/assets/images and replace icons that you want to change. Keep in mind that images have to be called same as original images (favicon.ico, icon-114.png, icon-144.png, icon-57.png and icon-72.png)

Comments

In minto theme we have implemented Disqus comment system. More on how to use disqus read here. Follow these steps to configure disqus for your website:

  1. First of all you have to register your profile at Disqus here.
  2. When you have finished step one go to Disqus and create forum on Disqus for your website. Click here to do that.
  3. At this point you have your forum registered at Disqus, now you have to copy your disqus forum shortname.
  4. Go to content/themes/minto/post.hbs and edit that file on line 37. You have to replace forum-shortname with your forum shortname.

Google analytics

If you use google analytics to track your sites traffic you don't have to be worried about its implementation. We have done almost everything for you. To finish google analytics configuration for your site follow these steps:

  1. Get tracking code for your website, it should look something like this UA-XXXX-Y.
  2. Now open this file, content/themes/minto/default.hbs
  3. After second step you have to edit that file on line 51. Replace UA-XXXX-Y with your google analytics tracking code.

Google fonts

If you don't like Minto theme fonts combination you can create yours. Google fonts is awesome service to do that.

  1. Pick your font on http://google.com/fonts
  2. In bottom right corner you have Quick-use button, click on it, choose styles and character sets that fits you the best.
  3. At this point you have to open content/themes/minto/default.hbs file.
  4. In that file find line 28. Replace {{! Google fonts goes here }} text with google fonts import code like this one <link href='http://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>
  5. Finally open content/themes/minto/assets/css/style-custom.css file and change font-family where you want to. It is very important to have knowledge about HTML/CSS before editing style-custom.css.

Homepage

In this section we will explain to you how to change homepage text below main title.

  1. Firstly go to content/themes/minto/index.hbs and open it in your favorite text editor.
  2. By default we have 2 columns with classes left-column and right-column. In column with class left-column you put content that will be shown in left column and the same thing goes for right column. Below that 2 columns we have full width column. As you can se we are using bootstrap grid classes. More info about how to use bootstrap grid read here.
  3. We have created few button styling classes for you. When you use <a href="#"><a> you can add class button and combine it with classes such as blue-yellow, twitter, linkedin and underline.
